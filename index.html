Stereo

Your Friendly Music Player

Stereo is a javascript music player.

Controls

These are the controls:

  1. Buttons for controlling the playback
  2. A label for displaying track info
  3. Time display to show time
  4. Progress bars, for load and play progress

All elements are created dynamically from a default or customized template.

The internal playlist can be created in two ways: through the JS API or by using a jQuery plugin. The jQuery plugin looks for matching links and/or <audio> elements. The urls from the href and src attributes are added to the internal playlist. The urls can either be relative to a Stereo.options.urlRoot url, or absolute (including http://), or a mix of both.

History

If history is enabled, all links on the page will be hijacked and ajax-loaded, to prevent playback from stopping when navigating on the page. Optionally all external links and PDF files can be set to open in a new window/tab. This will only happen when the player is on:
